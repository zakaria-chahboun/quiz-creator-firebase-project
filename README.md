
## Quiz Creator with Firebase


#### Clone and Install:
```bash
git clone https://github.com/zakaria-chahboun/quiz-creator-firebase-project.git
cd quiz-creator-firebase-project

npm install
```

### Run test:
```bash
npm run dev
```
